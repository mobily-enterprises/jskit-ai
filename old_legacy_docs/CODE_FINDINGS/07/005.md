# Q7: Performance — attachment uploads burst on client

## Finding
- Location: `packages/chat/chat-client-runtime/src/shared/useChatRuntime.js:858-874`
- Behavior: `Promise.all` uploads all attachments at once.

## Impact
If `attachmentMaxFilesPerMessage` increases, this saturates bandwidth/UI thread.

## Minimal refactor
Queue uploads with a small concurrency limit (2–3) or serialize with `for...of`.
