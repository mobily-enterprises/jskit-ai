# Q4: Convenient lies — readdir failures treated as empty lists

## Finding
- Locations:
  - `scripts/framework/_descriptor-utils.mjs:137`
  - `packages/tooling/jskit/src/shared/index.js:433`
- Behavior: `readdir` failures return an empty list, masking permission/IO errors as “no files found.”

## Why risky
Silent failures cause missing descriptors without any signal; CI and dev workflows can look healthy while skipping inputs.

## Better error strategy
Log a warning or rethrow in non-test contexts to avoid silent skips; include the path and error details.
