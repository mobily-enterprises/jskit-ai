# Q3: Contract drift — transcript message pagination limits

## Finding
- Contract defaults/limits:
  - `packages/ai-agent/assistant-fastify-routes/src/shared/schema.js:32-35` (default 500, max 500)
  - `packages/workspace/console-fastify-routes/src/shared/schema.js:1214-1218` (default 100, max 500)
  - `packages/workspace/workspace-fastify-routes/src/shared/schemas/admin.schema.js:284-288` (default 100, max 500)
- Runtime: `packages/ai-agent/assistant-transcripts-core/src/shared/service.js:12-16` and `:108-114` enforce default 20 and max 200.

## Drift
Contracts allow page sizes up to 500 and advertise higher defaults, but runtime clamps to 20/200.

## Risk
Clients will request larger pages or assume higher defaults and silently get less data, causing pagination bugs and UX inconsistencies.

## Suggested fix
Align schemas to `{ defaultPageSize: 20, maxPageSize: 200 }` or raise runtime limits to match documented defaults.
