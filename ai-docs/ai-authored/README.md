# AI-Authored Docs

This directory is for AI-authored documentation that is not reproducible from a command.

Use this directory for files that:
- were drafted or heavily rewritten by an AI
- require review, correction, and improvement by an AI or a human
- cannot be regenerated deterministically from the codebase

Do not put command-generated inventory here.

That belongs in:
- `../autogen/`

Rule of thumb:
- `autogen/` means "rerun the build command"
- `ai-authored/` means "open the file and improve it"
