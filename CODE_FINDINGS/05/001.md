# Q5: Hidden global state — realtime event bus singleton

## Finding
- Location: `apps/jskit-value-app/src/platform/realtime/realtimeEventBus.js:1-39`
- Behavior: Module-level `listeners` Set implements a global event bus.

## Why risky
Listeners leak across app instances/tests, ordering is non-deterministic, and unrelated modules can observe each other’s events.

## Suggested fix
Export `createRealtimeEventBus()` and inject the bus into runtimes/stores; provide `dispose()` to clear listeners in tests.
