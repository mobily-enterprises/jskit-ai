# Q5: Hidden global state — realtime URL from build/runtime globals

## Finding
- Location: `apps/jskit-value-app/src/platform/realtime/realtimeRuntime.js:96-118`
- Behavior: Realtime URL derived from `import.meta.env.VITE_REALTIME_URL` or `window.location`.

## Why risky
Build-time env and global window drive runtime, making tests/SSR non-deterministic.

## Suggested fix
Accept a `realtimeBaseUrl` option from bootstrap and pass it explicitly.
