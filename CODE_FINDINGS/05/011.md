# Q5: Hidden global state — repository config singleton + global override

## Finding
- Location: `apps/jskit-value-app/config/index.js:255-273`
- Behavior: `repositoryConfig` singleton plus `globalThis.__JSKIT_TEST_REPOSITORY_CONFIG_OVERRIDE__`.

## Why risky
Test overrides leak between suites; non-local mutation of configuration.

## Suggested fix
Pass overrides explicitly to consumers; avoid global overrides in runtime code.
