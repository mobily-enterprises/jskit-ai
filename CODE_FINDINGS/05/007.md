# Q5: Hidden global state — realtime runtime singleton

## Finding
- Location: `apps/jskit-value-app/src/app/bootstrap/runtime.js:102-199`
- Behavior: Module-level `realtimeRuntimeInstance` enforces a single runtime globally.

## Why risky
