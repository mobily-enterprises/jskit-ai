# Q5: Hidden global state — browser error reporter singleton

## Finding
- Location: `apps/jskit-value-app/src/platform/observability/browserErrorReporter.js:8-157`
- Behavior: Module-level `installed`, `reportsInFlight`, `suppressReporting` plus global `window` listeners.

## Why risky
Once installed, subsequent app mounts can’t reconfigure or uninstall; tests leak listeners.

## Suggested fix
Expose `{ start, stop }` instance; track listeners and allow teardown; inject `fetch`.
