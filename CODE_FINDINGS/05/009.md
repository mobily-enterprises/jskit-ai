# Q5: Hidden global state — module runtime singletons

## Finding
- Locations:
  - `apps/jskit-value-app/src/modules/assistant/runtime.js:7-18`
  - `apps/jskit-value-app/src/modules/chat/runtime.js:10-24`
  - `apps/jskit-value-app/src/modules/social/runtime.js:7-14`
- Behavior: Module-level runtime singletons capture config and dependencies at import time.

## Why risky
SSR/tests share a singleton; configuration changes don’t apply; hidden subscriptions persist.

## Suggested fix
Export `create*Runtime()` wrappers and instantiate per app bootstrap.
