# Q9: Security — CLI path traversal / arbitrary file write in element eject

## Finding
- Location: `packages/tooling/app-scripts/src/shared/commands/elementCommon.js:69-79` accepts absolute and `../` target paths; `packages/tooling/app-scripts/src/shared/commands/elementEject.js:21-45` writes to the resolved path.

## Threat
If `--target` is untrusted, this can overwrite arbitrary files.

## Fix
Require targets to be inside `appRoot` by default and add an explicit escape hatch (`--allow-outside-root`) with a warning.
