# Q9: Security — dynamic import of runtime modules without containment checks

## Finding
- Location: `packages/tooling/app-scripts/src/shared/runtimeCommands.js:68-70`
- Behavior: resolves `relativePath` with `path.resolve(appRoot, relativePath)` and `import()`s it. Config can be overridden in `app.scripts.config.*`.

## Threat
A malicious repo can point to arbitrary files outside the app root (code execution).

## Fix
Reject absolute paths and enforce that resolved paths stay within `appRoot` unless an explicit `--allow-outside-root` flag is set.
