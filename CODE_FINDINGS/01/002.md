# Q1: Action ID shadow copies

## Finding 1: Assistant action IDs duplicated
- File/lines: `packages/ai-agent/assistant-fastify-routes/src/shared/controller.js:3`
- Source of truth should be: `packages/runtime/action-runtime-core/src/shared/actionIds.js`
- Shadow copy exists: `ASSISTANT_ACTION_IDS` string map in controller
- Why risky: Action IDs may diverge from the central registry, breaking action dispatch and access control.
- Suggested refactor: Import `ACTION_IDS` and replace `ASSISTANT_ACTION_IDS.*` usages with `ACTION_IDS.ASSISTANT_*`.
