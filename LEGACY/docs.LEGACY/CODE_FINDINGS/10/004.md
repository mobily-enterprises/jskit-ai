# Q10: Invariant not asserted — mount override keys must be valid

## Finding
- Locations:
  - `packages/web/web-runtime-core/src/shared/clientComposition.js:227-231` only applies overrides when key exists.
  - `apps/jskit-value-app/config/urls.js:1-4` defines overrides.

## Invariant assumed
All override keys are valid; typos never occur.

## Why risky
Unknown keys are silently ignored; expected overrides do nothing.

## Suggested fix
Validate override keys after mount resolution and warn/throw on unused keys. Add a test for bogus override key.
