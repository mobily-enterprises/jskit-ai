# Console Access + Permissions Flow Tutorial

Console is global-surface (not workspace-bound) and has its own membership/role model.

## 1) Console routes are surface-specific API endpoints

```js
// packages/workspace/console-fastify-routes/src/routes.js
{
  path: "/api/console/bootstrap",
  method: "GET",
  auth: "required",
  handler: controllers.console?.bootstrap
}
```

## 2) Controller uses action executor only

```js
// packages/workspace/console-fastify-routes/src/controller.js
async function bootstrap(request, reply) {
  const payload = await executeAction(actionExecutor, {
    actionId: CONSOLE_ACTION_IDS.BOOTSTRAP_READ,
    request
  });
  reply.code(200).send(payload);
}
```

Same pattern applies for members/invites/settings/billing-admin actions.

## 3) Console access context resolution

```js
// workspace-console-service-core/src/services/consoleAccess.service.js
async function resolveRequestContext({ user }) {
  const userId = parsePositiveInteger(user?.id);
  ...
  const membership = await consoleMembershipsRepository.findByUserId(userId);
  const activeMembership = membership && membership.status === "active" ? membership : null;
  const permissions = activeMembership ? resolveRolePermissions(activeMembership.roleId) : [];
  ...
  return { membership: mapMembershipSummary(activeMembership), permissions, hasAccess: Boolean(activeMembership), pendingInvites };
}
```

## 4) Permission guards

```js
async function requireConsoleAccess(user) {
  const context = await resolveRequestContext({ user });
  if (!context.hasAccess) {
    throw new AppError(403, "Forbidden.");
  }
  return context;
}

async function requirePermission(user, permission) {
  const context = await requireConsoleAccess(user);
  if (!hasPermission(context.permissions, permission)) {
    throw new AppError(403, "Forbidden.");
  }
  return context;
}
```

## 5) Action definitions are console-surface-only

```js
// workspace-console-service-core/src/actions/console.contributor.js
{
  id: "console.member.role.update",
  channels: ["api", "internal"],
  surfaces: ["console"],
  permission: [CONSOLE_MANAGEMENT_PERMISSIONS.MEMBERS_MANAGE],
  idempotency: "optional",
  async execute(input, context) {
    ...
  }
}
```

App/admin assistant tool discovery does not include console-surface actions.

## 6) Root-user protection

`ensureRootMutationAllowed(...)` prevents non-root users from mutating root member state.

That guard lives in console access service and is called by console management flows.
