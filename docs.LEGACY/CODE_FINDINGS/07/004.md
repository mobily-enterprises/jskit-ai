# Q7: Performance — unbounded concurrency ensuring workspace settings

## Finding
- Location: `packages/workspace/workspace-service-core/src/shared/services/workspace.service.js:449-486`
- Behavior: `Promise.all` over `missingWorkspaceIds` with `ensureWorkspaceSettingsForWorkspace` per ID.

## Impact
Users with many workspaces can trigger a burst of DB writes, causing load spikes.

## Minimal refactor
Add bulk `ensureWorkspaceSettingsForWorkspaceIds` or process in chunks with a small concurrency cap.
